# Sorting Algorithms

Implementations of classic sorting algorithms in Python. Each algorithm
has its own folder containing the implementation, a runnable `example.py`,
and a dedicated README with a full explanation and complexity analysis.

| Algorithm | Time (avg) | Space | Stable | Folder |
|-----------|------------|-------|--------|--------|
| Bubble Sort | O(n²) | O(1) | Yes | [bubble-sort](bubble-sort) |
| Selection Sort | O(n²) | O(1) | No | [selection-sort](selection-sort) |
| Insertion Sort | O(n²) | O(1) | Yes | [insertion-sort](insertion-sort) |
| Merge Sort | O(n log n) | O(n) | Yes | [merge-sort](merge-sort) |
| Quick Sort | O(n log n) | O(log n) | No | [quick-sort](quick-sort) |

More sorting algorithms (e.g. Heap Sort) will be added over time — this
folder is updated weekly.
