# Searching Algorithms

Implementations of classic searching algorithms in Python. Each algorithm
has its own folder containing the implementation, a runnable `example.py`,
and a dedicated README with a full explanation and complexity analysis.

| Algorithm | Time (avg) | Space | Folder |
|-----------|------------|-------|--------|
| Binary Search | O(log n) | O(1) | [binary-search](binary-search) |

More searching algorithms will be added over time — this folder is
updated weekly.
