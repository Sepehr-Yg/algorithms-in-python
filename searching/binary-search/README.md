# Binary Search

Binary Search is an efficient searching algorithm that works on sorted arrays.

## How It Works

Binary Search repeatedly divides the search space into two halves until the target value is found or the search space becomes empty.

## Requirements

The input array must be sorted.

## Time Complexity

O(log n)

## Space Complexity

O(1)

## Example

Array:

[1, 3, 5, 7, 9, 11, 13]

Target:

7

Output:

3
